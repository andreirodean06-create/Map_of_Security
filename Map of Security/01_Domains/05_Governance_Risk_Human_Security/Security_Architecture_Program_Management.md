## Important Problems

* **No ownership:** controls exist on paper but nobody maintains them.
* **Security added too late:** security reviews happen after the system is already built.
* **Conflicting priorities:** usability, cost, deadlines, and security compete.
* **Open problem:** building security programs that improve real security, not only documentation.
